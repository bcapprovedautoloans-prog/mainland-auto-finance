# Mainland Automotive

Mainland Automotive is a modern auto finance website built to help customers across British Columbia get pre-approved for vehicles quickly and easily.

## Features

- Fast online vehicle pre-approvals
- Works for all credit types
- $0 down financing options
- Mobile-friendly responsive design
- Lead capture system
- Real-time urgency + approval UI
- Vehicle category selector
- Customer reviews section
- Prize wheel / engagement system
- Optimized landing page structure for high conversion

## Tech Stack

- HTML5
- CSS3
- JavaScript
- EmailJS integration

## Live Services

- Province-wide delivery across BC
- Car, SUV, truck, and van financing
- Trade-in and sell-your-car options

## Deployment

This project is hosted using GitHub and can be deployed through:

- Netlify
- GitHub Pages
